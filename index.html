Study Flashcards

What is HTML?
HTML is the structure of a webpage.
What is CSS?
CSS styles and designs web pages.
What is JavaScript?
JavaScript adds interactivity to websites.
What is a variable?
A container that stores data values.
What is a function?
A reusable block of code.
What is Flexbox?
A CSS layout system for alignment.
What is a loop?
Code that repeats until a condition is met.
What is a tag?
HTML keywords like <div> or <p>.
